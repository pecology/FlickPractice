# Flick Practice - フリック入力練習ゲーム

スマートフォンのフリック入力を練習するためのウェブアプリケーションです。

## 機能

- **60秒間のタイピングゲーム**: 制限時間内にできるだけ多くの文字を正確に入力
- **2つの入力モード**:
  - **ひらがなモード**: 日本語フリック入力の練習
  - **アルファベットモード**: 英語フリック入力の練習
- **2つのゲームモード**:
  - **デイリー**: 同じ日は同じ文字列が出題される
  - **プラクティス**: ランダムな文字列で練習
- **詳細な統計**:
  - CPM (Characters Per Minute)
  - 正確性
  - 苦手文字の分析
  - スコア推移グラフ

## 開発

### 必要環境

- Node.js 18+
- npm

### セットアップ

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

## 技術スタック

- TypeScript
- Vite
- 純粋なHTML/CSS（フレームワーク不使用）

## フリック入力の仕組み

スマートフォンの標準的なフリック入力と同じ操作：
- **タップ**: 中央の文字を入力
- **上フリック**: 上に表示された文字を入力
- **下フリック**: 下に表示された文字を入力
- **左フリック**: 左に表示された文字を入力
- **右フリック**: 右に表示された文字を入力
